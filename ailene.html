Lessons that I learned

I enjoyed learning HTML and CSS in Introduction to Computing. HTML helped me build the structure of a webpage, while CSS allowed me to style it, making the design more appealing. It was fun to see how small changes in code could transform the layout and colors. The challenge came with applying CSS for positioning, margins, and responsiveness across different screens. It took a lot of trial and error to get the design right on both desktop and mobile. Despite the difficulty, this experience deepened my appreciation for the creativity in web design.

Lessons that I learned

In Introduction to Computing, I learned about the fundamental components of a computer, such as the CPU, memory, and storage devices, and how they work together. I also explored how the operating system manages hardware and software resources to ensure smooth functionality. I then delved into algorithms and problem-solving, the history of computers, different types of software, and the importance of networking through the internet. Finally, I learned HTML and CSS, which taught me how to structure and style web pages. This course gave me a solid foundation in computing and introduced me to the basics of web development.
