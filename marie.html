Lessons that I learned

My Introduction to Computing class opened my eyes to how the CPU, memory, and storage devices work together, with the Operating System acting as a mediator to keep everything running smoothly. Learning algorithms taught me to break down problems step by step, a skill I’ve found helpful in many areas of life. Exploring HTML and CSS sparked my creativity, showing me how coding can bring structure and design together in exciting ways.

Overcoming Challenges

I really enjoyed working with HTML and CSS in my Introduction to Computing class because it felt like bringing ideas to life—building a webpage’s structure with HTML and then transforming it with CSS into something visually exciting. What fascinated me the most was how even small tweaks in the code could completely change the look and feel of the page, and experimenting with layouts and colors was so much fun. At the same time, I struggled with understanding CSS positioning and making designs responsive across different screens, which often led to a lot of trial and error. Looking back, though, those challenges are what made me appreciate the art and complexity of web design even more—it’s incredibly satisfying to turn a messy layout into something polished and functional.
